HTML Summary

HTML (HyperText Markup Language) is the heart of every webpage. It defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning.

  • HTML uses "elements" to describe (annotate) content HTML elements typically
  • You can also have void (empty) elements like images
  • You can also configure elements with atributes

Learn more about all available HTML elements on the MDN HTML element reference.
